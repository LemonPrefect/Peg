# Peg
MOVED TO https://github.com/LemonPrefect/Peg.ts
Peg is a brief CLI for DogeCloud OSS based on the web requests.

## Usage
Clone this repo with
```bash
git clone git@github.com:LemonPrefect/Peg.git
```
and use like
```bash
python peg.py --help
```
